Local HTTP-Server for Testing:
Install https://www.python.org/downloads/
Open CMD on Windows or Terminal on MAC
Change to directory of the index.html file by typing: cd "Path\To\Folder\Containing\Index\"
Start HTTP Server by typing:python -m http.server
CTRL-C stops the server.
The index.html file should be accessible through the browser via localhost:8000/index.html
